# Python-UDPScanner

### Description
The project use UDP to scan your subnet.

### How to use?
```sh
	Parameter:

	-h --host   : Your computer IP.
	-t --target : The target you want to scan in subnet. Example:192.168.1.0/24

	example: sudo python UDPScanner.py -h 192.168.1.103 -t 192.168.1.0/24

```
